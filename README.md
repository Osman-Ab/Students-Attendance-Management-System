# 🎓 ASTU Pre-Engineering Attendance System

A simple **C++ console-based Attendance Management System** using file handling.

## ✨ Features
- Add and delete students
- Mark attendance
- View attendance records
- Update attendance status
- Store data using text files

## 🛠 Technologies Used
- C++
- File Handling
- Structures
- Functions

## 📂 Files
- `students.txt` → Stores student information
- `attendance.txt` → Stores attendance records

## 🚀 How to Run

```bash
g++ main.cpp -o attendance
./attendance
```
